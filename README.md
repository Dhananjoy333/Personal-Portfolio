# 🌐 Personal Portfolio Website

A modern and interactive personal portfolio website built using **Next.js**, showcasing my projects, skills, and background with smooth animations and engaging UI.

---

## 🚀 Features

- ⚡ **Hero Section**  
  Eye-catching introduction with dynamic typing effect using **Typed.js**

- 🧭 **Header & Footer**  
  Clean navigation and structured layout for better user experience

- 👨‍💻 **About Section**  
  Brief introduction about me, my journey, and interests

- 🎓 **Education Section**  
  Animated timeline showcasing my academic background using **Framer Motion**

- 📂 **Projects Section**  
  Display of my work with descriptions and links

- 📬 **Contact Section**  
  Easy way to get in touch

- 🎬 **Smooth Animations**  
  Implemented using **Framer Motion** for a modern feel

---

## 🛠️ Tech Stack

- **Framework:** Next.js  
- **Animations:** Framer Motion  
- **Typing Effect:** Typed.js
